About HN Hidden Gems

Discovering quality content before it hits the front page

The Origin Story

This project was born from a personal frustration: submitting a quality "Show HN" post that received zero traction simply because the account had only 1 karma point. Despite being a genuinely useful project, it was buried under hundreds of other submissions.

We realized that Hacker News has a hidden bias toward established users. Great content from new or low-karma accounts often gets overlooked, not because it lacks quality, but because it lacks the social proof of an established reputation.

"What if we could find the hidden gems before they become obvious?"

How It Works

Data Collection
  • Monitors HN's new posts feed in real-time
  • Uses official HN Firebase API (no rate limits!)
  • Real-time collection of all new posts
  • Tracks author karma and account age
Quality Analysis
  • AI-powered content evaluation
  • Technical depth assessment
  • Originality and innovation scoring
  • Problem-solving potential analysis
  • Advanced spam filtering

Key Features

Live Hidden Gems Feed

Real-time discovery of quality posts from low-karma accounts

Hall of Fame

Track gems that later became successful

Success Metrics

Measure discovery accuracy and timing

Multiple Notifications

RSS, email, Discord, and Slack integration

Technology Stack

Backend
  • Python 3.11+
  • Flask web framework
  • SQLAlchemy ORM
  • OpenAI GPT for analysis
  • Redis for caching
APIs & Data
  • HN Firebase API
  • SQLite/PostgreSQL
  • GitHub API integration
  • RSS feed generation
  • Email notifications

Our Mission

We believe that great ideas shouldn't be overlooked just because they come from new or low-karma accounts. Innovation often comes from unexpected places, and the best solutions sometimes come from people who are just starting their journey in the tech community.

By surfacing these hidden gems early, we hope to:

  • Give deserving projects the attention they merit
  • Help new developers and entrepreneurs get discovered
  • Create a more equitable platform for innovation
  • Build a community that values quality over reputation
